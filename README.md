# Costa (Un)Installer
![License: BSD 3-Clause](https://img.shields.io/badge/License-BSD%203--Clause-blue.svg)

## Install Costa Client (on Linux)
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/costa-security/install/HEAD/install.sh)"
```


## Uninstall Costa Client (on Linux)
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/costa-security/install/HEAD/uninstall.sh)"
```
